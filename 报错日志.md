## 1. `(trapped) error reading bcrypt version` + `ValueError: password cannot be longer than 72 bytes`

**时间**：2026-05-23

**现象**：注册时报 500 Internal Server Error，浏览器显示 "Internal S... is not valid JSON"

**原因**：`passlib` 和 `bcrypt 4.1+` 不兼容。passlib 很久没维护了，新版本 bcrypt 删除了 `__about__` 属性，且内部 `detect_wrap_bug` 测试使用了超 72 字节的字符串。

**解决**：`pip install bcrypt==4.0.1` 降级 bcrypt 到 4.0.1

**相关文件**：`app/routers/auth.py` 第 51 行

## 2. `AttributeError: type object 'datetime.datetime' has no attribute 'UTC'`

**时间**：2026-05-23

**现象**：登录时报 500，注册成功但登录炸了

**原因**：`datetime.UTC` 是 Python 3.11 新增的。本机 Python 版本低于 3.11，没有这个属性。

**解决**：
- 导入：`from datetime import datetime, timedelta, timezone`
- 用法：`datetime.now(timezone.utc)` 代替 `datetime.now(datetime.UTC)`

**相关文件**：`app/routers/auth.py` 第 6 行（导入）、第 33 行（使用）
